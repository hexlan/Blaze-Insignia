# BlazeInsignia

A browser-based tactical RPG inspired by Fire Emblem, built with TypeScript and Vite.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

Opens the game at `http://localhost:5173`

### Build

```bash
npm run build
```

Output is generated in the `dist/` directory. Runtime assets (images, audio, levels) are copied automatically via `scripts/copy-data.js`.
